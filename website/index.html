Get Started

Install in 2 Minutes

Click through each step — it's easier than you think.

Download the Extension

Grab the ZIP from GitHub — one click.

Download ZIP

Extract the ZIP

Right-click the .zip → "Extract All..."

Inside you'll see an AQWikiTools-main folder — that's the one you'll load in the next steps.

Open Extensions Page

Type chrome://extensions in your address bar.

Menu More tools Extensions

Enable Developer Mode

Flip the toggle in the top-right corner.

Click to try!

Load Unpacked

Click "Load unpacked" → select the AQWikiTools-main folder.

Pick the folder that has manifest.json inside it — not the outer wrapper folder.

Sync Your Inventory

Visit account.aq.com while logged in. The extension syncs automatically.

Syncing items...

You're All Set!

Visit aqwwiki.wikidot.com and enjoy! Open the popup for settings & Farm Tracker.
