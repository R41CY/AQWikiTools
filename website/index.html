AQWikiTools

A Chrome extension that supercharges the AQW Wiki with item previews, calculators, inventory tracking, dark mode, and more.

Download Extension Install Guide View Features

Features

Everything you need to enhance your AQW Wiki browsing experience.

Extension Popup

Quick access to inventory sync status, settings, dark mode toggles, and the Farm Tracker from one clean popup.

Settings & Active Boosts

Toggle dark mode, image previews, and character page settings. See your active server boosts and access quick links to the Farm Tracker and AQW Wiki.

Dark Mode & Owned Items

Full wiki dark mode with owned item highlighting — items you own are color-coded with bank icons right on the page.

Hover Image Preview

Hover over any item to see a detailed preview with location, rarity, price, and description instantly.

Quest Calculator

Track quest progress with a live progress bar, showing needed vs. owned materials and what's still missing.

Farm Tracker — To Drop

Browse all unowned drop items in a filterable card grid with rarity filters, search, and drop source info.

Item Detail & Source Preview

Click any item for a detailed view with description and drop sources. Hover a source to preview the monster image and location.

Farm Tracker — To Merge

See all unowned merge shop items grouped by shop, with NEEDED badges and ownership progress per shop.

In Bank View

View all your banked items in a beautiful card grid with IN BANK badges, rarity tags, and source info.

Collection Stats

Track your overall completion with donut charts for Non-Rare, AC, and Seasonal item categories.

Character Page Compare

When viewing another player's character page, an [Owned] tag appears next to items you also own — quickly compare inventories at a glance.

Installation Guide

Install AQWikiTools in Chrome in under 2 minutes.

1

Download the Extension

Download the AQWikiTools project as a .zip file. Click the button below or visit the GitHub repository.

Download ZIP
2

Extract the ZIP File

Right-click the downloaded .zip file and select "Extract All..." (or use 7-Zip / WinRAR). Extract it to a folder you can find easily, like your Desktop or Documents. Do not skip this step — Chrome cannot load a .zip directly.

3

Open Chrome Extensions

Open Google Chrome and navigate to chrome://extensions in the address bar. You can also get there via Menu > More tools > Extensions.

4

Enable Developer Mode

In the top-right corner of the extensions page, flip the "Developer mode" toggle to ON. This allows you to load custom extensions.

5

Load the Extension

Click "Load unpacked" in the top-left. Browse to the extracted folder and select the folder that contains manifest.json (this is the root of the extension). Click "Select Folder".

6

Sync Your Inventory

Go to account.aq.com/AQW/Inventory while logged in to your AQW account. The extension will automatically detect and sync all your inventory and bank items. Wait for the page to fully load.

7

Browse the Wiki

Visit aqwwiki.wikidot.com and enjoy hover previews, merge/quest calculators, owned item markers, dark mode, and more! Open the extension popup to access settings and the Farm Tracker.
