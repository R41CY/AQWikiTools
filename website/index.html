Features

Everything you need to enhance your AQW Wiki browsing experience.

UI

Extension Popup

Quick access to inventory sync status, settings, dark mode toggles, and the Farm Tracker from one clean popup.

UI

Settings & Active Boosts

Toggle dark mode, image previews, and character page settings. See your active server boosts at a glance.

UI

Dark Mode & Owned Items

Full wiki dark mode with owned item highlighting — color-coded with bank icons right on the page.

Tool

Hover Image Preview

Hover over any item to see a detailed preview with location, rarity, price, and description instantly.

Tool

Quest Calculator

Track quest progress with a live progress bar, showing needed vs. owned materials and what's still missing.

Tracker

Farm Tracker — To Drop

Browse all unowned drop items in a filterable card grid with rarity filters, search, and drop source info.

Tracker

Item Detail & Source Preview

Click any item for a detailed view with description and drop sources. Hover a source to preview the monster.

Tracker

Farm Tracker — To Merge

See all unowned merge shop items grouped by shop, with NEEDED badges and ownership progress per shop.

Tracker

In Bank View

View all your banked items in a beautiful card grid with IN BANK badges, rarity tags, and source info.

Tracker

Collection Stats

Track your overall completion with donut charts for Non-Rare, AC, and Seasonal item categories.

Tool

Character Page Compare

When viewing another player's character page, an [Owned] tag appears next to items you also own.

Get Started

Install AQWikiTools in Chrome in under 2 minutes. Click through each step.

Download the Extension

Grab the ZIP file from GitHub — one click, that's it.

Download ZIP

Extract the ZIP

Right-click the downloaded .zip → "Extract All..."

Chrome can't load a .zip directly — you need the extracted folder.

Open Extensions Page

In Chrome, go to chrome://extensions

Menu More tools Extensions

Enable Developer Mode

Flip the "Developer mode" toggle in the top-right corner.

Click to try!

Load Unpacked

Click "Load unpacked" → select the extracted folder containing manifest.json

Make sure you select the root folder, not a subfolder.

Sync Your Inventory

Visit account.aq.com/AQW/Inventory while logged in. The extension syncs automatically.

Syncing items...

You're All Set!

Visit aqwwiki.wikidot.com and enjoy all the features. Open the extension popup for settings and the Farm Tracker.

100% Open Source

Zero Data Collection

The extension stores everything locally in your browser. No servers, no analytics, no tracking. Your inventory data never leaves your machine.

Minimal Permissions

Only requests storage (to save your settings) and access to artix.com (to check server boosts). Nothing else.
